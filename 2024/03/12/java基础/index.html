创建线程方式

  1. 继承Thread类，重写run方法。
  2. 实现runable接口。重写run方法。
  3. 实现callable接口。此方式可以获得返回值，其它不可以。
  4. 线程池创建

String,StringBuilder,StringBuffer

String不可变，StringBuilder线程不安全，速度快。StringBuffer线程安全，速度慢。

ArrayList和LinkedList区别

  1. ArrayList底层时数组，LinkedList底层时链表。
  2. ArrayList有下标更适合查找，LinkedList更适合增删。
  3. LinkedList还实现了Deque接口，能被当作队列。

copyOnWriteArrayList与ArrayList

ArrayList线程不安全，copyOnWriteArrayList线程安全。

跨域问题


方法上加注解或者通过配置实现跨域。

bean


Spring本身并没有对bean做线程安全的处理。

  1. 如果bean无状态，那就是线程安全的。
  2. 有状态线程不安全。

什么是Bean

通俗一点来说baen就是java对象，只不过是由容器创建，并且由容器管理对象生命周期。
Bean的本质就是Java中的对象，而Spring中的Bean其实就是对实体类的引用，来生产Java类对象，从而实现生产和管理Bean 。

@Transactiona

此注解用来声明事务，用在方法或者类上，值有事务级别和超时时间等，必须是被public修饰的，是被代理的类，否则不生效。

#{}与${}区别

#{}是预编译处理，是占位符可以防止sql注入。
${}是占位符，只会进行字符串替换，不安全。
